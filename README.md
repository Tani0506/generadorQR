# QR Generator JS

Página estática que genera códigos QR en el navegador.

## Uso
1. Abre [la demo](https://TU_USUARIO.github.io/qr-generator-js/) o clona y abre `index.html`.
2. Escribe texto o URL.
3. Pulsa «Generar» y descarga la imagen con clic derecho → «Guardar imagen como…».

## Stack
- [qrcode.js](https://github.com/soldair/node-qrcode) vía CDN (sin dependencias locales).
- 100 % cliente; sin backend.
